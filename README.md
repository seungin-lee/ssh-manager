# HOW TO USE
## config file
Please refer the '.ssh-manager' directory and make it on your home directory.
You can make multiple config file

### Example of config file (.ssh-manager.conf/.ssh-manager.conf.A)
```
username: travis
ip: ['192.168.45.131','192.168.45.132'] 

username: seungin
ip: ['100.1.4.2']
```
